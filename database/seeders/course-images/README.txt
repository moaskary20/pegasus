صور الدورات - Course Cover Images
====================================

ضع صور الغلاف المخصصة للدورات هنا لاستخدامها عند تشغيل الـ seeder.

أسماء الملفات المدعومة:
- 1.jpg, 2.jpg, 3.jpg ... (حسب ترتيب الدورة)
- programming.jpg, design.jpg, marketing.jpg, business.jpg (حسب التصنيف)
- أو اسم الدورة بالإنجليزية مثل: learn-programming-from-scratch.jpg

الصيغ المدعومة: jpg, png, webp

الحجم الموصى به: 800×450 بكسل أو نسبة 16:9

إذا لم تجد الصور هنا، سيتم إنشاء صور ملونة تلقائياً لكل دورة.
