# 📸 خطوات الاستخدام - صور توضيحية (نصية)

---

## 🎯 الخطوات الكاملة لإضافة اجتماع Zoom

### الخطوة 1: ادخل إلى لوحة التحكم

```
اذهب إلى: https://your-domain.com/admin
```

---

### الخطوة 2: اذهب إلى الدورات التدريبية

```
Menu: الدورات التدريبية (Courses)
        ↓
       اختر دورة موجودة
        ↓
       او اضغط إضافة جديدة
```

**ستجد:**
```
┌─────────────────────────────────┐
│       جميع الدورات             │
├─────────────────────────────────┤
│  اسم الدورة    │  الحالة │ عدد الدروس  │
├─────────────────────────────────┤
│  الدورة الأولى │  نشطة  │  5 دروس    │
│  الدورة الثانية│  نشطة  │  8 دروس    │
└─────────────────────────────────┘
```

---

### الخطوة 3: ادخل إلى الأقسام

```
بعد اختيار الدورة:
        ↓
    الأقسام (Sections)
        ↓
    اختر قسم او أضف جديد
```

**ستجد:**
```
┌────────────────────────────────┐
│    أقسام الدورة               │
├────────────────────────────────┤
│  رقم │ اسم القسم │ الدروس │    │
├────────────────────────────────┤
│  1   │  المقدمة   │  3    │ Edit│
│  2   │  الأساسيات │  5    │ Edit│
│  3   │  المتقدم   │  4    │ Edit│
└────────────────────────────────┘
```

---

### الخطوة 4: ادخل إلى الدروس

```
اضغط على: Edit (المحرر)
        ↓
    اضغط على: Lessons
        ↓
    ستجد قائمة الدروس
```

**ستجد:**
```
┌──────────────────────────────────────┐
│         دروس القسم                   │
├──────────────────────────────────────┤
│ # │ العنوان │ النوع │ Zoom │ تعديل  │
├──────────────────────────────────────┤
│ 1 │ الدرس 1│ فيديو│ لا  │ Edit   │
│ 2 │ الدرس 2│ نص   │ لا  │ Edit   │
│ 3 │ الدرس 3│ مختلط│ لا  │ Edit   │
└──────────────────────────────────────┘
```

---

### الخطوة 5: اضغط تحرير (Edit) أو إضافة (Add)

```
اضغط على: Edit (تحرير) لدرس موجود
أو
اضغط على: Add (إضافة) لدرس جديد
```

**سيفتح النموذج:**
```
┌──────────────────────────────────────┐
│      نموذج تحرير/إضافة الدرس        │
├──────────────────────────────────────┤
│ العنوان:        [____________]      │
│ الوصف:         [____________]      │
│ نوع المحتوى:    [اختر]             │
│ الفيديو:       [رفع ملف]           │
│                                    │
│ ⬇️ ⬇️ ⬇️ اسقط الى الأسفل ⬇️ ⬇️ ⬇️  │
│                                    │
│ ☐ إضافة اجتماع Zoom    ← اضغط ON  │
│                                    │
└──────────────────────────────────────┘
```

---

### الخطوة 6: فعّل خيار Zoom

```
ابحث عن:
    ☐ إضافة اجتماع Zoom
        ↓
    اضغط على الصندوق لتفعيله
        ↓
    سيتحول إلى:
    ☑ إضافة اجتماع Zoom ← مفعّل!
```

**بعد التفعيل ستظهر حقول جديدة:**
```
┌──────────────────────────────────────┐
│ ☑ إضافة اجتماع Zoom               │
├──────────────────────────────────────┤
│ 📅 موعد الاجتماع:                   │
│    [اختر التاريخ] [اختر الوقت]     │
│                                    │
│ ⏱️ مدة الاجتماع:                    │
│    [___60___] دقيقة                │
│                                    │
│ 🔐 كلمة مرور الاجتماع:              │
│    [____________] (اختياري)        │
│                                    │
└──────────────────────────────────────┘
```

---

### الخطوة 7: ملء تفاصيل الاجتماع

#### 7.1: اختر التاريخ والوقت

```
اضغط على: موعد الاجتماع
        ↓
    ستظهر نافذة تاريخ/وقت
        ↓
    حدد التاريخ: 2026-02-15
        ↓
    حدد الوقت: 14:30
        ↓
    اضغط: تأكيد (Confirm)
```

**النتيجة:**
```
📅 موعد الاجتماع: 2026-02-15 14:30
```

#### 7.2: حدد مدة الاجتماع

```
حقل مدة الاجتماع يحتوي على: 60
        ↓
    غيّره إلى المدة المطلوبة:
    • 60 دقيقة (ساعة واحدة) - افتراضي
    • 90 دقيقة (ساعة ونصف)
    • 120 دقيقة (ساعتان)
    • إلخ...
```

**الحد الأدنى:** 15 دقيقة  
**الحد الأقصى:** 480 دقيقة (8 ساعات)

#### 7.3: كلمة المرور (اختياري)

```
كلمة مرور الاجتماع: [حقل فارغ]
        ↓
    يمكن تركه فارغاً (اختياري)
        ↓
    أو اكتب كلمة مرور مثل: 123456
```

---

### الخطوة 8: احفظ النموذج

```
اسقط إلى أسفل النموذج
        ↓
    ابحث عن زر: حفظ (Save)
        ↓
    اضغط عليه
```

**سيظهر:**
```
⌛ جاري الحفظ...
        ↓
✅ تم الحفظ بنجاح!
        ↓
سيعود إلى قائمة الدروس
```

---

### الخطوة 9: تحقق من الإضافة

```
في قائمة الدروس الآن ستجد:
┌──────────────────────────────────────┐
│ # │ العنوان │ النوع │ Zoom │ تعديل  │
├──────────────────────────────────────┤
│ 1 │ الدرس 1│ فيديو│ لا  │ Edit   │
│ 2 │ الدرس 2│ نص   │ 📹  │ Edit   │ ← جديد!
│ 3 │ الدرس 3│ مختلط│ لا  │ Edit   │
└──────────────────────────────────────┘

📹 = اجتماع Zoom مرتبط!
```

---

## ✅ تم!

الآن الدرس لديه اجتماع Zoom مرتبط به:
- ✅ التاريخ والوقت محفوظة
- ✅ المدة محددة
- ✅ كلمة المرور (إن وجدت) محفوظة

---

## 🔍 عرض التفاصيل

### للعودة وتعديل الاجتماع:

```
1. اذهب إلى نفس الدرس
2. اضغط Edit
3. ستجد البيانات محفوظة:
   ☑ إضافة اجتماع Zoom
   📅 موعد الاجتماع: 2026-02-15 14:30
   ⏱️ مدة الاجتماع: 60 دقيقة
   🔐 كلمة مرور: 123456 (إن كتبت)
4. عدّل ما تريد
5. احفظ التغييرات
```

---

## 🔄 تعطيل Zoom

### إذا أردت حذف الاجتماع:

```
1. ادخل إلى تحرير الدرس
2. ابحث عن: ☑ إضافة اجتماع Zoom
3. اضغط عليه لتعطيله (سيصبح ☐)
4. سيختفي الاجتماع
5. احفظ
```
