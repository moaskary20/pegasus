## 📋 جميع حقول Zoom الآن

```
1️⃣ Toggle: ☐ إضافة اجتماع Zoom
   └─ (يفعّل باقي الحقول)

2️⃣ DateTimePicker: 📅 موعد الاجتماع
   └─ متى يكون الاجتماع؟

3️⃣ TextInput: ⏱️ مدة الاجتماع
   └─ كم دقيقة سيستغرق؟

4️⃣ TextInput: 🔐 كلمة مرور الاجتماع
   └─ كلمة مرور إضافية (اختياري)

5️⃣ TextInput: 🔗 رابط اجتماع Zoom
   └─ أين سيكون الاجتماع؟ (مطلوب)
```
